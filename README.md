# Lead Prediction using Machine Learning

## Project Overview
This project predicts whether a lead is likely to convert using supervised machine learning models.

**Dataset:**
-real world lead conversion dataset used for machine learning classification.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Workflow
- Data Cleaning
- Missing Value Handling
- Feature Encoding
- Model Training
- Model Evaluation
- prediction

## Models Used
- MultinomialNB
- Decision Tree
- Random Forest
- Support Vector Machine
- KNeighborsClassifier

## Results
The best-performing model achieved approximately 98% accuracy on the test dataset.
